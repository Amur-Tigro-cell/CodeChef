# CodeChef Solutions

This repository contains my solutions to various **CodeChef** programming problems. The solutions are written primarily in **C++** and are organized for easy reference and practice.

## 📂 Repository Structure

* Each file contains the solution to a single CodeChef problem.
* File names generally match the problem title or code.
* All solutions are tested against the official CodeChef test cases.

## 💻 Language

* C++17

## 🚀 How to Run

Compile the program:

```bash
g++ -std=c++17 filename.cpp -o output
```

Run the executable:

```bash
./output
```

Or on Windows:

```bash
output.exe
```
